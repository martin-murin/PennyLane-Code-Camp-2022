Challenge code

You must complete the fourier_decomp function to return a list with the Fourier frequencies and the real and imaginary parts of the Fourier coefficient associated with these frequencies.

The input layers_params of fourier_decomp specifies how many trainable circuit blocks we have in our circuit, how many basic entangler layers each block contains, and what are the parameters in the rotation within the entangler layers. The elements of the list layers_params are lists of lists of real numbers. The -th element of layers_params corresponds to the -th trainable circuit block. If this element is of length (i.e. it contains lists of numbers), it means that the -th trainable circuit block contains basic entangler layers.
